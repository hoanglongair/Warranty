# Warranty - Premium Web3 Freelance Marketplace

![Warranty](public/favicon.svg)

A next-generation freelance marketplace connecting Job Posters (Người Thuê) and Freelancers (Người Được Thuê) with Web3 technology.

## Features

- 🌐 **Multi-language Support** - English, Vietnamese, Chinese, French
- 🔐 **Smart Contract Escrow** - Secure payments protected by blockchain
- ⚡ **Instant Settlement** - Get paid in seconds with USDC, ETH, or USDT
- 🎨 **Premium UI/UX** - Modern glassmorphism design with aurora effects
- 📱 **Responsive Design** - Works on all devices
- 🔗 **Web3 Wallet Integration** - MetaMask, WalletConnect, Coinbase Wallet

## Tech Stack

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **State Management**: Zustand
- **Form Handling**: React Hook Form + Zod
- **Icons**: Lucide React

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start
```

## Project Structure

```

```
